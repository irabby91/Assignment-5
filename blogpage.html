What is DOM

The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects. Each node in this tree corresponds to a part of the document (such as an element, attribute, or piece of text), and you can manipulate these nodes with programming languages like JavaScript.

How does it work

1.Browser Reads HTML: When the browser loads an HTML page, it looks at all the tags and content, like div, h1, p, and turns them into a DOM tree.
2.JavaScript Can Access and Change the DOM: Find elements: You can use JavaScript to find parts of the page, like headings or paragraphs. Change content: You can change what these elements say or how they look. Add or remove elements: You can add new things or remove parts from the page.
3.Handling User Actions: The DOM also lets you respond to user actions (like clicks, typing, etc.). You can add event listeners to elements and do something when the user interacts.

What is a DOM event

A DOM event is an action that happens on a web page, like clicking a button, typing in a text box, or loading the page. JavaScript can respond to these events to make the page interactive.
